Sweet Moments Assistant

---

# 📌 Ідея проєкту

Sweet Moments Assistant — це консольний помічник для кондитерської або домашнього cake-бізнесу, який допомагає зберігати інформацію про клієнтів, важливі дати, сімейні події та вподобання.

Головна ідея проєкту — підтримка персоналізованої комунікації з постійними клієнтами.

Наприклад:

Анна щороку замовляє торт чоловіку та дитині.
Асистент допомагає не забути про важливі дати, нагадує про можливість оформлення замовлення та допомагає підтримувати теплий персоналізований сервіс.

Система також допомагає:

* нагадувати про майбутні сімейні свята
* зберігати вподобання клієнтів
* вести персоналізовані нотатки
* підтримувати постійну комунікацію з клієнтами
* пропонувати бонуси або подарунки для постійних клієнтів

Підтримуються міжнародні телефонні номери для клієнтів з різних країн Європи.

# 📦 Встановлення проєкту

Клонувати репозиторій:

```bash
git clone <repository_link>
```

Перейти до папки проєкту:

```bash
cd sweet-moments-assistant
```

Встановити залежності:

```bash
pip install -r requirements.txt
```

# ▶️ Запуск проєкту

```bash
python -m assistant.main
```

# 📌 Основні команди

## Контакти

```text
add [name] [+phone]
change [name] [old_phone] [new_phone]
phone [name]
all
```

## Email та адреса

```text
add-email [name] [email]
show-email [name]

add-address [name] [address]
show-address [name]
```

## День народження

```text
add-birthday [name] [DD.MM.YYYY]
show-birthday [name]
birthdays
```

## Нотатки та теги

```text
add-note [name] [text]
show-notes [name]

add-tag [name] [tag]
show-tags [name]
```

## Пошук

```text
find-contact [text]
find-note [text]
find-tag [text]
```

## Довідка

```text
help
exit
```

# 🧭 Інструкція для користувача

Вітаю! Це Sweet Moments Assistant — консольний помічник для кондитерської або домашнього cake-бізнесу.

Він допомагає зберігати інформацію про клієнтів, їхні контакти, дні народження, сімейні події, нотатки та теги.

Наприклад, якщо клієнтка Anna щороку замовляє торти для чоловіка та дитини, assistant допоможе зберегти цю інформацію, швидко знайти її та вчасно нагадати про важливу подію.

---

## ▶️ Як запустити програму

У терміналі потрібно ввести:

```bash
python -m assistant.main
```

Після запуску з’явиться головне меню з основними командами.

---

## 📌 Як подивитися всі команди

Якщо користувач не знає, що робити далі, потрібно ввести:

```text
help
```

Assistant покаже список доступних команд.

---

## 👤 Як додати клієнта

```text
add Anna +436601112233
```

Де:

* `add` — команда додавання клієнта
* `Anna` — ім’я клієнта
* `+436601112233` — номер телефону

Телефон має бути у міжнародному форматі:

* починатися з `+`
* містити 10–15 цифр

---

## 📧 Як додати email

```text
add-email Anna anna@gmail.com
```

Email має містити:

* `@`
* `.`

---

## 📍 Як додати адресу

```text
add-address Anna Vienna Center
```

Адресу можна писати кількома словами.

---

## 🎂 Як додати день народження клієнта

```text
add-birthday Anna 14.02.1990
```

Дата вводиться у форматі:

```text
DD.MM.YYYY
```

---

## 👨‍👩‍👧 Як додати сімейну подію

```text
add-occasion Anna Husband birthday
```

або:

```text
add-occasion Anna Son birthday
```

Це потрібно для того, щоб bakery могла пам’ятати важливі події клієнта та нагадувати про можливість замовлення торта.

---

## 📝 Як додати нотатку

```text
add-note Anna Orders cakes for husband and son
```

або:

```text
add-note Anna Loves chocolate cakes
```

У нотатках можна зберігати вподобання клієнта, історію замовлень або важливі деталі.

---

## 🏷️ Як додати тег

```text
add-tag Anna VIP
```

або:

```text
add-tag Anna chocolate
```

Теги допомагають швидко групувати клієнтів.

Наприклад:

* `VIP`
* `chocolate`
* `kids`
* `birthday`

---

## 🔍 Як знайти клієнта

```text
find-contact Anna
```

Пошук працює без урахування регістру, тому можна писати:

```text
find-contact anna
```

---

## 🔍 Як знайти нотатку

```text
find-note husband
```

Assistant знайде клієнтів, у нотатках яких є це слово.

---

## 🔍 Як знайти тег

```text
find-tag VIP
```

або:

```text
find-tag vip
```

---

## 📋 Як показати всіх клієнтів

```text
all
```

Assistant покаже всі збережені контакти.

---

## 🎂 Як перевірити найближчі дні народження

```text
birthdays
```

Assistant покаже дні народження, які будуть найближчим часом.

---

## ⚠️ Що буде, якщо ввести неправильні дані

Якщо ввести неправильний номер:

```text
add Petro 123
```

Assistant повідомить:

```text
Phone must start with + and contain 10-15 digits.
```

Якщо ввести неправильний email:

```text
add-email Anna wrongemail
```

Assistant повідомить:

```text
Invalid email format.
```

Програма не закривається після помилки, а продовжує працювати.

---

# 🗑️ Як видалити контакт

Щоб видалити клієнта, потрібно ввести:

```text
delete Anna
```

де:

* `delete` — команда видалення
* `Anna` — ім’я клієнта

Після цього assistant видалить контакт із системи.

---

## ⚠️ Якщо контакту не існує

Наприклад:

```text
delete Petro
```

Assistant повідомить:

```text
Contact not found.
```

Програма не закривається після помилки та продовжує працювати.

---

# 🚪 Як завершити програму

Щоб завершити роботу та зберегти дані, потрібно ввести:

```text
exit
```

або:

```text
close
```
